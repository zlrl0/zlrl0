  #### 1. PickO(픽오) 홈페이지 제작
<a href="https://github.com/zlrl0/PickO">
   <img alt=Scoit"" width="400" src="https://github.com/user-attachments/assets/55be7685-508a-4475-86b9-a7a22b93aeb9"/>
</a>

< PickO : Pick your OTT 🎥📺 >
<br/>
<br/>

Picko는 다양한 OTT 플랫폼(넷플릭스, 티빙, 왓챠 등)에서 제공하는 콘텐츠를 한 웹사이트에서 간편하게 찾아볼 수 있으면 좋지 않을까?하는 생각에서 시작한 프로젝트입니다.

<br/>
● OTT 플랫폼 통합 : 여러 플랫폼의 콘텐츠를 한눈에 확인하고 구독료를 비교
<br/>
●개별 콘텐츠 : 콘텐츠 관련 정보(줄거리, 추천 콘텐츠, 시청 가능한 OTT 플랫폼)를 확인
<br/>
● 커뮤니티 : 사용자 간 콘텐츠 추천 및 리뷰를 공유할 수 있는 커뮤니티 기능
<br/>
<br>
Picko는 여러 OTT 플랫폼을 오가며 생기는 불편함을 해소하고, 사용자 맞춤형 콘텐츠 추천을 통해 누구나 편하게 사용할 수 있는 홈페이지를 만들고자 노력했습니다.
<br/>
<br/>

  #### 2. 충치디펜스 게임 제작
  <a href="https://github.com/zlrl0/CavityDefense">
   <img alt=Scoit"" width="400" src="https://github.com/user-attachments/assets/144106e3-8abe-4187-beed-7cf00a278bc3"/>
  </a>
  
  < 충치 디펜스 >
  <br>
  <br/>
  충치 디펜스는 어린이들에게 충치 예방의 중요성을 재미있고 시각적으로 전달하면 어떨까?하는 생각에서 시작된 프로젝트입니다. 단 음식을 피하며 깨끗한 치아를 유지하는 과정에서 충치의 위험성을 자연스럽게 학습할 수 있습니다.
  <br/>
  <br/>
How to play?
  <br/>
  떨어지는 단 음식을 화살표 키를 이용해 피하며 점수를 획득
  <br/>
  플레이어의 목숨(3개)이 모두 소진되면 게임 종료
  <br/>
  조작법 : 화솰표 키로 좌우 이동
  <br/>
  <br/>
  간단한 게임을 통해 어린이들이 재미있게 즐기며 충치 예방의 중요성을 배우고 치아 건강에 대한 긍정적인 습관을 형성하는데 도움이 됩니다.
  <br/>
  자세한 내용을 다음을 확인하세요.
  <br/>
  (Repository: https://github.com/zlrl0/CavityDefense)
